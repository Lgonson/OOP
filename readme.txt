The project is based on the tutorial of free code camp, url: 
https://www.youtube.com/watch?v=Ej_02ICOIgs&t=1158s

1) The main class for the project is the item class, represented in the file
item.py
2) phone.py and keyboard.py are two child classes from item.py.
3) main is the framework implemented to work with these classes.
4) The csv file instances.csv contains a few instances to use as example
for the OOB project.



5) The file named Object Oriented Programming.pdf is a Latex summary of the
tutorial.
